# auto_versio_diretorio README
1. json.dump() vs json.dumps()
(Serve para salvar ou transformar algo em JSON)

json.dump(): Salva o dicionário do Python direto para dentro de um arquivo .json físico.

json.dumps(): Só transforma o dicionário em uma linha de texto (string) dentro do próprio código, sem criar arquivo nenhum.

2. json.load() vs json.loads()
(Serve para ler um JSON e trazer para o Python)

json.load(): Abre e lê as informações direto de um arquivo .json que está no seu computador.

json.loads(): Lê um texto em formato JSON que já está salvo em uma variável de texto (string) no seu código.
